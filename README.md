# Insurance Tech Audit Analytics Lab

Technology audit engagement simulator for a synthetic Canadian insurance environment.

## Simulated Company
**NorthStar Life Insurance**  
Cloud-first insurance company running policy administration, claims, finance reporting, and customer portals across Azure and AWS.

## Objective
Support a technology audit engagement by analyzing synthetic enterprise data, testing IT and security controls, documenting exceptions, and presenting audit-ready findings.

## Focus Areas
- Internal audit analytics
- IAM
- Cloud security
- Infrastructure security controls
- Vulnerability management
- Financial services / insurance context

## Repository Modules
- Audit planning and governance
- Synthetic insurance datasets
- SQL audit tests
- Power BI dashboard pack
- Formal audit workpapers
- Framework mapping
