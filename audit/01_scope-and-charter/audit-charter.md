# Audit Charter and Scope Memo

## Engagement Title
Technology Audit Analytics Review - Identity, Cloud, and Infrastructure Controls

## Entity
NorthStar Life Insurance

## Background
NorthStar Life Insurance is a cloud-first Canadian insurer operating policy administration, claims, finance reporting, and customer-facing platforms across Azure and AWS. This repository simulates a technology audit engagement designed to assess selected IT and security controls using synthetic enterprise data and audit analytics.

## Audit Objective
Assess whether key identity, cloud, infrastructure, and vulnerability-management controls are designed and operating effectively to reduce technology and cyber risk in a financial-services environment.

## Scope
This simulated audit focuses on:
- User access and entitlements
- Privileged and admin accounts
- MFA coverage
- Service account governance
- IAM role assignments
- Asset inventory / CMDB coverage
- Vulnerability management and patching status
- Firewall / network rule inventory
- Cloud security posture findings
- Backup and disaster recovery control signals
- Issue remediation tracking

## Key Audit Questions
1. Are privileged and high-risk access rights appropriately restricted and monitored?
2. Are critical identity controls such as MFA and account ownership consistently enforced?
3. Are high-risk vulnerabilities identified and remediated within expected timelines?
4. Are cloud and infrastructure configurations aligned to control expectations?
5. Are control exceptions documented, escalated, and tracked to remediation?

## In-Scope Platforms
- Azure
- AWS
- Identity and access management data sources
- Vulnerability and patch management data sources
- Infrastructure / network control inventories
- Backup and disaster recovery evidence logs

## Expected Outputs
- Risk assessment summary
- Control universe
- Risk-control matrix
- Evidence request list
- SQL-based control tests
- Dashboard-ready findings
- Sample audit issues and management actions

## Notes
This is a synthetic portfolio project designed to demonstrate audit-defensible analysis, formal documentation, and control-testing logic in a financial-services technology audit setting.
